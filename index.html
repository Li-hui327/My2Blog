hexo 搭建博客经验分享

  1. 最开始我是参考博主小坏孩的博客制作的，传送门：Hexo 博客 搭建超详细教程，我在这里简单介绍一下步骤：
  • 1、安装 Git 以及连接 Github
    • Git 下载地址：https://git-scm.com/download 根据对应的版本下载安装即可
    • 安装，如果不想修改安装目录的话，一路 next 即可，修改安装目录后也是一路 next 即可
    • 检查 git 是否安装成功，在 cmd 窗口中，输入 git 显示如下：
    • Git 连接 Github，首先在 Github 上注册一个账号(有账号的可以先建一个仓库，之后什么也不做，留着之后用)
    • 可在桌面右键菜单点击 Git Bash Here，如下图： 接着在打开的窗口设置 user.name 和 user.email 配置信息：
    1. git config --global user.name " 你的 GitHub 用户名 " 回车
    2. git config --global user.email " 你的 GitHub 注册邮箱 " 回车
    • 然后生成 ssh 密钥文件
      ssh-keygen -t rsa -C " 你的 GitHub 注册邮箱 " 回车
    • 在 C:\Users\ 用户名.ssh 中找到 id_rsa.pub(用记事本打开)，将里面的所有内容全部复制出来
    • 在 https://github.com/settings/keys 点击 New SSH key, 如下：
    • Title 随便填写，Key 粘贴刚才复制的 key，然后点击 Add SSH key，验证一下是否连接成功
    • 在 Git Bash 中输入 ssh git@github.com 如果显示下图说明连接成功
    • 完
